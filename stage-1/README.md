# 🚀 Interactive User Profile Generator

A beginner-friendly Python project that demonstrates how to collect user input, process and clean text, convert data types, apply logical conditions, and display a formatted profile summary.

This project is designed to help learners practice the fundamentals of Python through a practical, real-world example.

---

## 📚 Learning Objectives

By completing this project, you will learn how to:

- Capture user input using the `input()` function.
- Convert user data with `int()`, `float()`, and `str()`.
- Clean and transform text using common string methods.
- Manipulate strings with concatenation, repetition, and indexing.
- Use logical operators (`and`, `or`, `not`) to control program flow.
- Format output using **f-strings** and the `.format()` method.
- Write clean, readable, and beginner-friendly Python code.

---

### Input & Type Casting
